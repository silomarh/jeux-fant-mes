# 👻 Fantômes

Deux jeux de mémoire jouables dans le navigateur, sans installation.

**[fantomes.fr](https://fantomes.fr)**

---

## Les jeux

### Mots Fantômes
Un mot est caché phonétiquement dans une phrase. 
Les sons se fondent entre les mots pour dissimuler le mot cible.
→ Mémoire auditive, attention phonologique.

### Clavier Fantôme
Un clavier sans les lettres. Tape les mots affichés de mémoire musculaire.
La difficulté augmente avec le score.
→ Mémoire procédurale, rapidité.

---

## Stack

- HTML / CSS / JS vanilla
- Supabase (phrases Mots Fantômes)
- Vercel (hébergement)
